# ReadMD 阶段 03 开发记录：稳定性与文档

日期：2026-05-24

## 阶段目标

本阶段目标是在阶段 02 的本地备忘录 MVP 基础上，补齐一部分日常使用稳定性能力，并提供面向用户的使用说明和下载安装说明。

## 已完成

1. 设置持久化
   - 大字模式会保存。
   - 字号设置会保存。
   - 行距设置会保存。
   - 下次打开应用时自动恢复。

2. 自动草稿保护
   - 编辑内容发生变化时保存本地草稿。
   - 新建备忘录但未保存时也会保存草稿。
   - 应用重新打开时自动恢复上次未保存草稿。
   - 保存成功或放弃修改后清除草稿。

3. 行距调节
   - 新增“行宽”和“行窄”操作。
   - 配合“字大”“字小”满足基础阅读调节需求。

4. 草稿状态反馈
   - 有未保存草稿时显示“草稿已保护”提示。

5. 本地单元测试
   - 新增 `MarkdownPreviewTest`。
   - 覆盖文内搜索匹配统计的基础逻辑。

6. 用户文档
   - 新增 [使用说明](../user-guide.md)。
   - 新增 [下载安装说明](../installation.md)。
   - 更新 README 文档入口和项目状态。

## 验证记录

已执行：

```powershell
.\gradlew.bat :app:testDebugUnitTest :app:assembleDebug
```

结果：

```text
BUILD SUCCESSFUL
```

说明：

- 第一次在普通沙箱中运行时，Gradle Wrapper 尝试访问网络下载分发包，被网络限制拦截。
- 按权限流程重新执行后使用缓存/可用环境完成验证。
