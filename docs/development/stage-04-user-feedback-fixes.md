# ReadMD 阶段 04：真机反馈修复记录

日期：2026-05-24

## 反馈问题

1. 大字模式下按钮文字可能换行后挤在一起，典型场景是“关闭大字”按钮。
2. “字体”“行距”滑动条调整后，界面文字和间距没有明显变化。
3. 打开文件后还需要点“编辑”，不符合备忘录类应用的直觉。
4. 导入文件、点击“阅读”、打开 Markdown 文件时仍可能闪退，疑似 Markdown 预览渲染链路不稳。

## 处理方案

1. 将按钮高度改为根据大字模式和文字长度自动抬高，并允许按钮文字最多两行显示。
2. 将字体滑动条绑定到首页、文档页、设置页、按钮、最近文件列表和编辑框字体。
3. 将行距滑动条绑定到页面区块间距和 Markdown 预览行距。
4. 移除“阅读/编辑”切换按钮，文件打开后直接进入可编辑状态。
5. 保留下方 Markdown 预览区域，但把预览器改成“先解析为安全块，再由 Compose 渲染”，减少特殊 Markdown 内容导致闪退的风险。
