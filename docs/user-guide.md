# ReadMD 使用说明

ReadMD 是一款 Android 本地 Markdown 备忘录应用。它不需要登录账号，不上传文档，主要用于在手机上打开、阅读、编辑、保存和导出 Markdown 文件。

当前版本的核心体验是“先阅读，再编辑”：打开文件后默认进入阅读页，给 Markdown 预览更大的显示空间；需要修改内容时，再点击“进入编辑模式”。

## 1. 首页

打开 ReadMD 后会进入首页。

首页主要操作：

- 点击“导入”打开本地 `.md`、`.markdown` 或 `.txt` 文件。
- 点击“新建”创建一个新的 Markdown 备忘录。
- 在“搜索”框中筛选最近打开的文件。
- 点击最近文件卡片，重新打开对应文件。
- 点击“清空”删除最近打开记录。
- 点击右上角“设置”调整字体、行距、大字模式和夜间模式。

说明：

- ReadMD 使用 Android 系统文件选择器，不会主动扫描整个手机文件夹。
- 最近记录只保存文件 URI、展示名、打开时间、可写状态和内容摘要，不会复制或上传你的文件。

## 2. 阅读 Markdown

导入文件或从最近记录打开文件后，ReadMD 默认进入阅读页。

阅读页包含：

- 顶部文件名。
- 大面积 Markdown 预览区。
- “进入编辑模式”按钮。
- 右上角“首页”和“设置”入口。

阅读页支持手势缩放：

- 两指放大，可增大字号。
- 两指缩小，可减小字号。
- 缩放只作用于阅读页预览，不影响编辑框输入。

当前版本支持常见 Markdown 预览：

- 标题、段落、引用。
- 加粗、斜体、删除线、行内代码。
- 有序列表、无序列表、任务列表。
- 代码块、分割线、表格。
- 部分 HTML 内容的基础显示。

当前限制：

- 暂不支持 PDF 或图片导出。
- 暂不保证复杂数学公式、Mermaid 图表等 Obsidian 扩展语法的完整渲染。
- 默认只处理本地文档内容，不主动加载网络资源。

## 3. 编辑 Markdown

在阅读页点击“进入编辑模式”后，会进入编辑页。

编辑页包含：

- “保存”“另存”“导出”三个操作按钮。
- 一个大面积 Markdown 原文编辑框。
- 顶部“阅读”按钮，用于返回阅读页查看渲染效果。

使用建议：

- 编辑时直接输入 Markdown 原文。
- 输入法弹出后，当前页面会优先把空间留给编辑框。
- 文件名旁或标题中出现 `*` 时，表示当前内容有未保存修改。
- 编辑过程中会自动保留本地草稿，降低误关应用造成内容丢失的风险。
- 在编辑页点击“保存”或“另存”后，应用会自动回到阅读页，方便立即查看渲染效果。

## 4. 保存、另存和导出

### 保存

点击“保存”会尝试写回当前文件。

- 如果当前文件可写，会直接保存。
- 如果当前文件不可写，或当前内容来自新建草稿，会进入系统另存流程。

### 另存

点击“另存”会创建一个新的 Markdown 文件，并把新文件作为当前编辑文件。

适合场景：

- 原文件不可写。
- 想保留原文件，同时保存一份修改后的副本。
- 新建备忘录第一次保存到手机文件夹。

### 导出

点击“导出”会把当前 Markdown 原文导出为新的 `.md` 文件。

说明：

- 导出不会替换当前正在编辑的文件。
- 当前版本导出的是 Markdown 文件，不是 PDF、JPG 或 PNG。

## 5. 新建备忘录

1. 在首页点击“新建”。
2. 应用会创建一个新的 Markdown 草稿，并进入编辑页。
3. 输入内容后点击“保存”或“另存”。
4. 在 Android 系统文件选择器中选择保存位置。

建议：

- 文件名建议以 `.md` 结尾，例如 `日记.md`、`会议记录.md`。
- 新建内容保存前属于应用内草稿，请及时另存到你容易找到的文件夹。

## 6. 设置与阅读辅助

点击右上角“设置”会打开设置面板。

当前设置项：

- 字体：通过滑动条调整界面和正文的字体大小。
- 行距：通过滑动条调整阅读和编辑时的行距、区块间距。
- 大字模式：点击“大字模式”开启，再次点击“关闭大字”关闭。
- 夜间模式：点击“夜间模式”进入深色阅读界面；按钮会切换为“日间模式”，再次点击可返回日间模式。

说明：

- 默认是日间模式。
- 大字模式和夜间模式互相独立，可以组合使用。
- 设置会自动保存，下次打开应用仍然生效。
- 夜间模式采用深底浅字，目标是降低夜间阅读刺激并避免黑底黑字。

## 7. 草稿保护

当你编辑内容但还没有保存时，ReadMD 会保存本地草稿。

适用场景：

- 切到后台后应用被系统回收。
- 不小心关闭应用。
- 编辑新建备忘录但还没保存到文件。

说明：

- 草稿只保存在本机应用私有数据中。
- 保存成功后，草稿会被清除。
- 如果重新打开应用时检测到未保存草稿，会尝试恢复。

## 8. 返回操作

ReadMD 已接入系统返回手势和返回键。

规则如下：

- 在编辑页按返回，会先回到阅读页。
- 在阅读页按返回，会返回首页。
- 在设置面板按返回，会先关闭设置面板。

## 9. 隐私说明

ReadMD 当前版本：

- 不需要账号登录。
- 不上传文档。
- 不支持云同步。
- 不默认加载网络图片。
- 不做密码保护或本地加密。

请注意：

- 如果你把文件保存到第三方网盘目录，或通过 Android 系统分享给其他应用，相关行为由系统或第三方应用处理。
