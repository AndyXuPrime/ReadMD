# ReadMD 使用说明

ReadMD 是一款 Android 本地 Markdown 备忘录应用。它不需要登录账号，不上传文档，主要用于在手机上打开、阅读、编辑、保存和导出 Markdown 文件。

## 1. 打开文件

1. 打开 ReadMD。
2. 点击“打开”。
3. 在 Android 系统文件选择器中选择 `.md`、`.markdown` 或 `.txt` 文件。
4. 文件打开后会进入阅读模式，并显示在最近打开列表中。

说明：

- ReadMD 使用 Android 系统文件选择器，不会扫描整个手机文件夹。
- 如果文件没有写入权限，界面会显示“需另存”，此时可以阅读和编辑，但保存时需要另存为新文件。

## 2. 新建备忘录

1. 点击“新建”。
2. 应用会创建一个新的 Markdown 草稿。
3. 输入内容后点击“保存”或“另存”。
4. 在系统文件选择器中选择保存位置。

建议：

- 文件名建议以 `.md` 结尾，例如 `日记.md`、`会议记录.md`。

## 3. 阅读 Markdown

打开文件后默认进入阅读模式。当前版本支持基础 Markdown 预览：

- 标题
- 段落
- 引用
- 有序/无序列表
- 任务列表
- 代码块
- 分割线
- 简单表格行展示

当前限制：

- 本地图片预览尚未完成。
- 链接点击尚未完成。
- 加粗、斜体、删除线、行内代码目前以清洗后的普通文本展示。
- 复杂表格、数学公式、Mermaid 图表暂不支持。

## 4. 编辑和预览

1. 点击“编辑”进入 Markdown 文本编辑模式。
2. 修改内容。
3. 点击“预览”查看未保存内容的阅读效果。
4. 如果不满意，可以再次点击“编辑”继续修改。
5. 如果想撤销未保存修改，点击“放弃”。

注意：

- 文件名后出现 `*` 表示当前有未保存修改。
- 未保存内容会自动保存为本地草稿，重新打开应用时会尝试恢复。

## 5. 保存、另存和导出

### 保存

点击“保存”会尝试写回当前文件。

- 如果当前文件可写，会直接保存。
- 如果当前文件不可写或是新建草稿，会自动走另存流程。

### 另存

点击“另存”会创建一个新的 Markdown 文件，并将它作为当前编辑文件。

### 导出

点击“导出”会把当前 Markdown 原文导出为新的 `.md` 文件。

说明：

- 导出不会改变当前正在编辑的文件。
- 当前版本暂不支持 PDF 和图片导出。

## 6. 最近打开

打开文件后，ReadMD 会记录最近打开文件，最多保存 10 条。

你可以：

- 点击最近文件快速重新打开。
- 点击“清空”删除最近记录。

说明：

- 最近记录只保存文件 URI、展示名、打开时间和可写状态。
- ReadMD 不复制、不上传你的文件。

## 7. 文内搜索

在阅读模式下，可以在“文内搜索”输入关键词。

当前版本会显示匹配数量。

后续计划：

- 匹配高亮
- 上一个/下一个跳转
- 自动滚动定位

## 8. 大字模式和阅读调节

ReadMD 提供基础适老化阅读能力：

- 点击顶部“大字”切换大字模式。
- 点击“字大”增大字号。
- 点击“字小”减小字号。
- 点击“行宽”增大行距。
- 点击“行窄”减小行距。

这些设置会自动保存，下次打开应用仍然生效。

## 9. 草稿保护

当你编辑内容但还没有保存时，ReadMD 会保存本地草稿。

适用场景：

- 切到后台后应用被系统回收。
- 不小心关闭应用。
- 编辑新建备忘录但还没保存到文件。

说明：

- 草稿只保存在本机应用私有数据中。
- 保存成功或放弃修改后，草稿会被清除。

## 10. 隐私说明

ReadMD 当前版本：

- 不需要账号登录。
- 不上传文档。
- 不支持云同步。
- 不默认加载网络图片。
- 不做密码保护或本地加密。

请注意：

- 如果你使用 Android 系统分享、导出或保存到第三方网盘目录，相关行为由系统或第三方应用处理。
